## 🎨 按钮设计展示

### 按钮样式

```
正常状态:
┌──────────────┐
│ 🔗 查看官方  │  ← 绿色渐变背景，白色文字
└──────────────┘

悬停状态:
┌──────────────┐
│ 🔗 查看官方  │  ← 稍微放大，发出光晕
└──────────────┘  ✨ (缩放效果 + 阴影)

表格按钮:
┌────┐
│查看│  ← 更小的版本，适配表格
└────┘
```

### 按钮颜色含义
- **绿色** (#28a745 → #20c997) = 安全、可信、官方
- **高对比度** = 易于识别和点击
- **光晕效果** = 交互反馈，提高用户体验

## 📞 后续支持

### 如果用户想添加新的官方数据源

编辑 `frontend/js/main.js` 中的 `OFFICIAL_DATA_SOURCES`:

```javascript
// 添加新的数据源
'YourMarket': {
    title: '市场名称',
    url: 'https://official-website.com',
    description: '数据描述'
}
```

### 如果链接无法打开

1. 检查网络连接
2. 检查浏览器是否阻止了弹窗
3. 手动复制链接到浏览器地址栏
4. 稍后重试（官方网站可能暂时无法访问）

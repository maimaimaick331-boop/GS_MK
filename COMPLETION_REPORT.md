## 📋 API服务总览

### 13个API端点

#### 数据采集 (1)
```
POST   /api/collect               手动触发数据采集
```

#### COMEX数据 (2)
```
GET    /api/comex/warehouse       获取库存历史数据
GET    /api/comex/latest          获取最新库存数据
```

#### ETF数据 (2)
```
GET    /api/etf/holdings          获取ETF持仓历史
GET    /api/etf/latest            获取最新ETF数据
```

#### 价格数据 (3)
```
GET    /api/price/latest          获取最新价格（全市场）
GET    /api/price/all             获取价格历史数据
GET    /api/price/by-market/<m>   获取特定市场价格
```

#### 分析数据 (2)
```
GET    /api/analytics             获取分析数据
GET    /api/analytics/summary     获取分析摘要
```

#### 系统信息 (3)
```
GET    /api/health                健康检查
GET    /api/logs                  获取采集日志
GET    /api                       API信息
```
